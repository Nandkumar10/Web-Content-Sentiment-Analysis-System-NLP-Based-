# 📊 Web Content Sentiment Analysis System (NLP-Based)

An NLP-based system that extracts textual data from web articles and performs sentiment analysis and readability assessment using custom linguistic features and statistical metrics.

---

## 🧠 About

This project automates the process of web content extraction and text analysis by:

- Scraping article data from URLs  
- Cleaning and preprocessing text  
- Performing sentiment analysis using custom dictionaries  
- Computing readability and linguistic metrics  

The system is designed to transform raw unstructured web content into meaningful insights, making it useful for content analysis, business intelligence, and research applications.

---

## 🚀 Features

- Automated web scraping from URLs  
- Text preprocessing using NLP techniques  
- Custom sentiment analysis (positive/negative scoring)  
- Polarity and subjectivity calculation  
- Readability metrics (Fog Index, sentence length, complexity)  
- Scalable pipeline for multiple documents  

---

## 🧱 Tech Stack

- Python  
- Pandas  
- NLTK  
- BeautifulSoup  
- Requests  
- Regex (Text Cleaning)  

---

## 🛠 How It Works

1. Load input dataset containing URLs  
2. Extract article content using web scraping  
3. Clean and preprocess text:
   - Tokenization  
   - Stopword removal  
   - Noise removal  
4. Apply sentiment analysis:
   - Positive & negative word matching  
   - Score calculation  
5. Compute readability metrics:
   - Sentence length  
   - Complex word percentage  
   - Fog Index  
6. Generate structured outputs for analysis  

---

## 📊 Metrics Calculated

### Sentiment Metrics
- Positive Score  
- Negative Score  
- Polarity Score  
- Subjectivity Score  

### Readability Metrics
- Average Sentence Length  
- Percentage of Complex Words  
- Fog Index  
- Syllable Count
